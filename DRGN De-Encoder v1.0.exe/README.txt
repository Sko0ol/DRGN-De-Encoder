DRGN De-Encoder v1.0 : How to use 

Autor: Sko0ol
Github: https://github.com/Sko0ol/DRGN-De-Encoder


Nachdem das Program gestartet wurde, kann man zwischen Encoden|1| und Decoden|2| wählen. 


Encoden wird die Datei codieren, also aus einem normal, lesbaren Text (z.B. "Hallo, das ist ein Test") 
einen nicht lesbaren Text machen (z.B. "Pfqqb, gfi xie vxz Evie"). 

Decoden wird die Datei decodieren, also aus dem nicht lesbaren, codierten Text (z.B. "Pfqqb, gfi xie vxz Evie")
einen lesbaren Text machen (z.B. "Hallo, das ist ein Test").


Nachdem man Encoden|1| oder Decoden|2| gewählt hat, muss man den Namen der Datei eingeben,
welche codiert bzw decodiert werden soll. Man muss den Namen in diesem Format eingeben "ExampleFile.txt".
Wenn man den Namen falsch schreibt, die Datei nicht existiert oder man ".txt" vergisst, wird ein Error erscheinen!
Groß- und Kleinschreibung wird ignoriert.


Wenn die Datei erfolgreich geladen wurde, kann man das Encoding Protokoll auswählen.
Encoding Protokolle:

LTADL (Letter-To-A-Different-Letter) wird jeden Buchstaben in einen anderen Buchstaben Umwandeln.
LTN   (Letter-To-Number)		Noch nicht hinzugefügt!
LTC   (Letter-To-Character)		Noch nicht hinzugefügt!

Groß- und Kleinschreibung wird ignoriert.

Wenn man am Anfang Decoden|2| gewählt hat, muss man das Encoding Protokoll auswählen, welches benutzt wurde 
um die Datei zu codieren.


Nachdem das Programm die Datei Erfolgreich codiert hat, kann man zwischen [EXIT] und [RETURN] auswählen.
[EXIT] wird das Programm schließen. [RETURN] wird das Programm neustarten 
Groß- und Kleinschreibung wird ignoriert.
